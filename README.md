# version-control

## 项目简介

- 本项目主要是为了写一篇版本控制的教程。通过此教程能让读者对项目控制有一定的认识，能了解到版本控制的概念，意义和工具。在教程中通过计算器项目实例，从零开始演示如何在项目中加入版本控制功能以及如何将项目存储到 GitHub 仓库。
  
## 项目目录结构

```python
docs\       # 存放教程
pic\        # 存放教程中所使用的图片
test_code\  # 存放教程中所提到的计算器项目的代码

```
